About the Developer

Welcome! I am an independent developer creating fun, engaging mobile experiences. Battle Bites is my latest project — an interactive discussion game where players decide who wins in head-to-head animal matchups.

About Battle Bites

Battle Bites encourages friendly debate by letting users vote on which animal would win in hypothetical battles. With over 30+ custom-illustrated animals and a playful visual style, the game is built to spark conversation.
